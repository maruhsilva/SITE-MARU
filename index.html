COMO FUNCIONA?

    1

  • Contato

    O cliente entra em contato conosco através dos nossos canais de comunicação informando qual projeto gostaria de orçar;

    • 2

  • Briefing

    Nesse momento o cliente recebe um documento para preenchimento contendo perguntas pertinentes para o tipo de projeto escolhido;

    • 3

  • Valores

    O orçamento é feito e entregue ao cliente contendo todos os passos e tempo de desenvolvimento do projeto;

    4

  • Envio

    Nessa etapa o cliente envia todos os arquivos e documentos necessários para o início do desenvolvimento do projeto;

    • 5

  • Entrega

    Após o desenvolvimento em conjunto com o cliente, e do tempo estipulado, é feito a entrega projeto desenvolvido.
